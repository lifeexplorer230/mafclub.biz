## ❌ КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. GEO (Generative Engine Optimization)

**Проблема:** Контент не структурирован для AI-моделей (ChatGPT, Claude, Gemini, Perplexity)

**Последствия:**
- AI-модели не могут найти и процитировать ваш сайт
- Потеря трафика из AI-ассистентов (растущий источник)
- Нет ответов на вопросы пользователей через AI

**Решение:**
- ✅ Добавить JSON-LD разметку (Organization, Event, Service)
- ✅ Создать FAQ секцию с Schema.org разметкой
- ✅ Структурировать контент для AI (списки, таблицы, четкие ответы)
- ✅ Создать ai-info.json файл

---

### 2. Структурированные данные (Schema.org)

**Проблема:** Минимальная разметка Schema.org

**Что отсутствует:**
- ❌ Organization schema
- ❌ Event schema (важно для мероприятий!)
- ❌ Service schema
- ❌ FAQPage schema
- ❌ BreadcrumbList schema
- ❌ Review schema

**Текущая разметка:**
- Только ImageObject в галерее (17 вхождений)

**Решение:**
- Добавить все недостающие Schema.org типы

---

### 3. Отсутствие FAQ

**Проблема:** Нет раздела FAQ с ответами на типичные вопросы

**Последствия:**
- AI-модели не могут дать ответы о вашем сервисе
- Потеря Featured Snippets в Google
- Пользователи не находят быстрые ответы

**Решение:**
- Создать FAQ секцию с минимум 7-10 вопросами
- Добавить FAQPage Schema.org разметку

---

### 4. Социальные сети

**Проблема:** Отсутствуют Twitter Card метатеги

**Последствия:**
- Некрасивые превью при шаринге в Twitter/X
- Потеря CTR из социальных сетей

**Решение:**
- Добавить twitter:card метатеги
- Добавить дополнительные OG теги (og:locale, og:site_name)

---

### 5. Производительность

**Проблема:** Большой размер JavaScript

**Текущий размер:**
- JavaScript: 544 KB
- Изображения: 1.9 MB
- CSS: 120 KB
- Fonts: 456 KB

**Последствия:**
- Медленная загрузка на мобильных устройствах
- Низкий PageSpeed Insights score
- Негативное влияние на SEO

**Решение:**
- Конвертировать изображения в WebP (-30-50% размера)
- Минификация JavaScript
- Lazy loading для изображений
- Code splitting

### 8. robots.txt

**Проблема:** robots.txt не настроен для AI-краулеров

**Отсутствуют директивы для:**
- GPTBot (OpenAI)
- CCBot (Common Crawl для AI)
- anthropic-ai (Claude)
- Google-Extended (Gemini)

**Решение:**
- Добавить User-agent для всех AI краулеров
- Разрешить индексацию (Allow: /)
