# Инструкция по размещению на GitHub Pages

## Шаг 1: Создайте репозиторий на GitHub

1. Перейдите на https://github.com/new
2. Введите название репозитория: `mafclub.biz`
3. Оставьте репозиторий **публичным** (для бесплатного GitHub Pages)
4. **НЕ** добавляйте README, .gitignore или лицензию (они уже есть)
5. Нажмите **Create repository**

## Шаг 2: Загрузите код в репозиторий

Выполните следующие команды в терминале из директории проекта:

```bash
# Добавьте remote репозиторий (замените YOUR_USERNAME на ваш GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/mafclub.biz.git

# Загрузите код
git push -u origin main
```

## Шаг 3: Настройте GitHub Pages

1. Перейдите в настройки репозитория: `Settings` → `Pages`
2. В разделе **Source** выберите:
   - Branch: `main`
   - Folder: `/ (root)`
3. Нажмите **Save**

## Шаг 4: Дождитесь деплоя

GitHub Pages автоматически развернет ваш сайт. Это может занять 1-5 минут.

Ваш сайт будет доступен по адресу:
```
https://YOUR_USERNAME.github.io/mafclub.biz/
```

## Шаг 5: (Опционально) Настройте кастомный домен

Если вы хотите использовать домен `mafclub.biz`:

1. В настройках репозитория GitHub → Pages → Custom domain введите: `mafclub.biz`
2. Добавьте DNS записи у вашего регистратора домена:
   ```
   A record: @ → 185.199.108.153
   A record: @ → 185.199.109.153
   A record: @ → 185.199.110.153
   A record: @ → 185.199.111.153
   CNAME record: www → YOUR_USERNAME.github.io
   ```
3. Дождитесь обновления DNS (может занять до 48 часов)

## Проверка локально

Перед загрузкой на GitHub можно проверить сайт локально:

```bash
# Запустите простой HTTP сервер
python3 -m http.server 8000

# Откройте в браузере
# http://localhost:8000
```

## Обновление сайта

Для обновления контента:

```bash
# Внесите изменения в файлы
# Затем:
git add .
git commit -m "Описание изменений"
git push origin main
```

GitHub Pages автоматически обновит сайт в течение нескольких минут.

## Возможные проблемы

### Сайт не открывается
- Проверьте, что GitHub Pages включен в настройках
- Подождите 5-10 минут после первого деплоя
- Убедитесь, что репозиторий публичный

### Не загружаются стили/изображения
- Проверьте пути в HTML (должны быть относительными)
- Убедитесь, что все файлы загружены в репозиторий

### 404 ошибка
- Проверьте URL: должен быть `https://USERNAME.github.io/mafclub.biz/`
- Убедитесь, что branch выбран правильно (main)
