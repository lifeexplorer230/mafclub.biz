#### 5. Оптимизировать изображения (WebP)

**Зачем:** Уменьшить размер на 30-50%, ускорить загрузку

```bash

# Установить webp (если не установлен)
sudo apt-get update
sudo apt-get install webp

# Конвертировать все JPEG в WebP
for file in *.jpeg *.jpg; do
  cwebp -q 85 "$file" -o "${file%.*}.webp"
done

# Проверить результат
```
